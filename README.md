# goIRL

Tech events in India worth showing up to — hackathons, meetups, workshops where you'll learn something *and* meet your people. Even if you come alone.

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Stack

- Next.js 16 (App Router, Turbopack)
- TypeScript + Tailwind v4
- Cookie-based mock auth (Supabase / Google OAuth coming in Phase 2)
- Seed data in `src/data/events.json`

## Deploy on Vercel

```bash
vercel --prod
```
